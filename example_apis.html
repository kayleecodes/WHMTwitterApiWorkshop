Weather APIs


Many apps like to incorporate weather data, especially if they involve going outside (for example, a hiking app which tells you the weather so you know what to wear).

OpenWeatherMap is a popular API for getting weather data, but it is just one of many.

Login with ______


Almost all social media networks (and Amazon) offer "login with" APIs. This allows some websites to remove friction from the signup process and creates more connections with that network.

Login with Amazon/Google/Facebook/Twitter. Find out more about Login With Amazon here.

Travel Booking

Many companies which aggregate travel data to show you different options are powered by APIs.

The aggregating sites call travel (airline/car rentals/hotels) APIs to get information about availability and then call a different set of APIs to do the booking.
