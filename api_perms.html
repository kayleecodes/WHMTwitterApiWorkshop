Permissions

The Twitter APIs require a set of credentials that you must pass with every request.

You must apply for access prior to our workshop. This is a simple process, but it takes 1-2 days.

The first step in the process is creating a twitter account, which is required (Sorry!).

When asked for your reasoning, you can say that you are learning about APIs and wish to practice.
