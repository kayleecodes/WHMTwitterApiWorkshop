2000

Salesforce launches an API. XML (eXtensible Markup Language) APIs were part of Salesforce from day one.

2002

Amazon launched Amazon.com Web Services allowing developers to incorporate Amazon.com content in their own websites

2004

Flickr launches an API so that users can easily share photos on their blogs and social networks

2006

Facebook launches their API

Twitter launches their API (due in large part to people creating rogue APIs)

Google launches their map API

AWS launches solely via APIs. Amazon did not launch a web interface for three more years.

2009

Apple launched the iPhone 3G, opening up entirely new world of mobile applications, one that APIs would be the driving force.
